# Step2 匹配滤波输出格式

## Step2当前状态

- 输入：extracted文件
- 输出：matched文件（独立的文件）

## Step2期望修改

- 输入：extracted文件
- 输出：matched文件（独立的文件）
- matched文件格式修改为：
  - ch1: 喇叭激励
  - ch2: mic响应
  - ch3: 匹配滤波结果

## matched文件ch3输出格式

| 通道 | 名称 | 内容 | 截取时间 | 截取长度 | chirp长度 | 响应窗口长度 | 响应窗口起始位置 |
|------|------|------|----------|----------|-----------|--------------|-----------------|
| ch1 | 喇叭激励 | chirp信号 | - | - | - | - | - |
| ch2 | mic响应 | mic信号 | - | - | - | - | - |
| ch3 | 匹配滤波结果 | 滤波输出 | 0s | 2s | PS:20ms, SV:25ms, SH:25ms, A0H:25ms, A0L:100ms | PS:50.6ms, SV:75ms, SH:75ms, A0H:105ms, A0L:1200ms | PS:0.0173s, SV:0.035s, SH:0.0382s, A0H:0.065s, A0L:0.2s |

## 各波型详细参数

| 波型 | chirp持续 | delay_min | 响应窗口长度 |
|------|-----------|-----------|--------------|
| PS | 20ms | 0.0173s | 50.6ms |
| SV | 25ms | 0.035s | 75ms |
| SH | 25ms | 0.0382s | 75ms |
| A0H | 25ms | 0.065s | 105ms |
| A0L | 100ms | 0.2s | 1200ms |
